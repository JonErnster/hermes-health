# Hermes Health

Public homepage for **Hermes Health** — a personal, single-user longevity dashboard that
unifies wearable data, bloodwork, nutrition, body composition, and DNA-informed longevity
guidance.

This repo hosts only the GitHub Pages site (homepage, privacy policy, terms, logo) used as
the app's domain for OAuth consent-screen configuration. The application itself runs locally
on the owner's machine and is not published here.

Site: https://jonernster.github.io/hermes-health/
